Virtual Pet (Alien)

Vertual-pet is an interactive user interface where user can create an organic or cyoorg aliens (pets). Then, players can take actions such as feed, work or get smart on signle or all pets. This game also tracks the status of aliens as player takes actions. Players also has an option to adapt or add aliens to shelter. This game also provides an ability to play as long as players like, and leave the game when they want.

Skills And Technologies used

  • Agile Methodology
  • Java OOP(Abstraction, Polymorphism, Inheritance and Encapsulation)
  • TDD

High Street Hospital

This project is an user interactive console app which allows user to print a list of employees personal information quickly. It also has an option to print all employees' work and pay statuses. Lastly, user can choose an employee and have them perform a task and see the results.

Skills And Technologies used

  • Agile Methodology
  • Java OOP(Abstraction, Polymorphism, Inheritance and Encapsulation)
  • TDD
